# Portfolio

A personal portfolio website showcasing my projects, skills, and experience.

## Features

- **Warm Neumorphic Design** — A modern, textured aesthetic with smooth animations
- **3D Flip Cards** — Interactive project showcase with hover-to-flip card effects
- **Masonry Layout** — Clean, staggered grid for project tiles
- **Contact Integration** — Formspree-powered contact form + Calendly booking
- **Responsive Design** — Fully responsive across devices

## Tech Stack

- HTML5, CSS3, JavaScript
- [Formspree](https://formspree.io/) for contact form submissions
- [Calendly](https://calendly.com/) for scheduling

## Getting Started

To run locally, serve the files with any static server:

```bash
python -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000) in your browser.
